# دليل المساهمة في awesome-mortakaz

أهلًا بك في مستودع **awesome-mortakaz**، وهو مستودع يهدف إلى جمع وعرض المشاريع والمبادرات التقنية العربية المميزة المرتبطة بمجتمع مرتكز.

نرحب بجميع المساهمات التي تساعد على إثراء المحتوى وتحسين جودة المستودع.

---

## كيفية المساهمة؟

1. قم بعمل Fork للمستودع.
2. أضف أو عدّل المحتوى وفق قواعد التصنيف المعتمدة.
3. تأكد من صحة الروابط والمعلومات.
4. أرسل Pull Request مع وصف واضح للتغييرات.

---

## طريقة التصنيف المتبعة

يتم تقسيم المحتوى حسب أسماء المساهمين أو الفرق، ثم تُدرج المشاريع التابعة لكل مساهم داخل قسمه.

مثال:

```md
# اسم المساهم

- [اسم المشروع](الرابط): وصف مختصر للمشروع.
```

ويُفضّل أن يكون الوصف:

* مختصرًا وواضحًا.
* يشرح الهدف الرئيسي للمشروع.
* لا يتجاوز سطرًا أو سطرين.

---

## كيفية إضافة مشروع جديد؟

أضف اسمك إلى الفهرس الرئيسي إن لم يكن موجودًا:

```md
- [اسمك](#اسمك)
```

ثم أضف قسمًا جديدًا:

```md
# اسمك

- [اسم المشروع](الرابط): وصف مختصر للمشروع.
```

مثال:

```md
# زكرياء العلوي

- [بيئة زيكو جي إس](الرابط): نظام متكامل بلغة جافاسكريبت لتطوير تطبيقات الويب.
```

---

## قواعد كتابة المشاريع

يجب أن يتضمن كل مشروع:

* اسم المشروع.
* رابط المشروع أو المستودع.
* وصف مختصر.
* رابط المصدر (GitHub أو GitLab) إن وجد.

---

## اقتراحات وتطوير المستودع

إذا كانت لديك أفكار لتحسين طريقة التصنيف أو تنظيم المحتوى أو إضافة أقسام جديدة، يمكنك فتح Issue ومناقشتها مع المجتمع قبل التنفيذ.

---

شكراً لكل من يساهم في إثراء مجتمع مرتكز ومشاركة المشاريع العربية المفتوحة المصدر.
